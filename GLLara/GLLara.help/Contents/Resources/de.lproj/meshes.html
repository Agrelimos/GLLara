Meshes

Ein Modell in GLLara besteht aus einem oderen mehren Meshes. Meshes sind Teilmodelle, die vom Grafiksystem mit den selben Einstellungen gezeichnet werden.

Hinweis: Meshes sind dabei völlig unabhängig von Bones. Ein Mesh enthält mehrere Bones, und die Teile des Modells, die ein Bone kontrolliert, können in mehreren unterschiedlichen Meshes liegen.

Sie finden die Meshes eines Objekts im Dokumentenfenster, unter dem Objekt im Unterpunkt ”Meshes”. Hier haben sie für jeden Mesh mehrere Einstellmöglichkeiten.

Sichtbarkeit

Sie können verschiedene Meshes sichtbar und unsichtbar machen. Damit können sie zum Beispiel verschiedene Zubehöritems wie Waffen auswählen.

Culling

Standardmäßig rendert GLLara die Hinterseiten von Objekten nicht. Hier können sie festlegen, welche Seite die Hinterseite sein soll. Sie können auch bestimmen, dass GLLara beide Seiten rendern soll, was besonders bei transparenten Objekten andere Ergebnisse erzielt.

Aussehen

Sie können das Aussehen ändern, in dem sie hier weitere Einstellungen ändern:

  • Texturen
  • Renderparameter
  • Shader
