Bilder exportieren

Sie können die aktuelle Ansicht eines Renderfensters als Bild exportieren. Im Vergleich mit einem Screenshot gibt es hier viel mehr Möglichkeiten, da sie das Bild auch mit hohen Auflösungen und transparentem Hintergrund exportieren können.

Zum Exportieren stellen sie sicher, dass das Renderfenster, dass sie exportieren wollen, im Vordergrund ist. Wählen sie dann im Menü “Ablage” den Punkt “Zu Bilddatei rendern…”.

Sie können hier beliebige Größen (bis maximal 50,000 Pixel Seitenlänge) auswählen und eine Reihe von Dateiformaten. Beachten sie, dass größere Bilddateien länger zum Speichern benötigen können, und nicht alle Programme sehr große Bilddateien fehlerfrei anzeigen können.

Hinweis: Es ergibt häufig bessere Ergebnisse, wenn sie das Bild in höherer Auflösung rendern, als sie benötigen, und es dann in einem Bildbearbeitungsprogramm verkleinern.

Wenn das Dateiformat Transparenz unterstütz (wie beispielsweise Portable Network Graphics bzw. PNG), dann wird der Hintergrund automatisch transparent. Das ermöglicht eine einfache Weiterbearbeitung in Bildbearbeitungsprogrammen.
