Exporting images

You can export the current view of a render window as an image. This gives you more options than a screenshot, since you can export the images in higher resolutions or with a transparent background.

To export, ensure that the render window you want to export is in the front. Then in the menu “File” select the item “Render to Image…”.

You can select arbitrary sizes (up to 50,000 pixels edge length) and a number of file formats. Note that larger files need longer for saving. Not all programs can open very large files without problems.

Note: There are often better results if you render the image at a larger size than you need, and then scale it down in an image editing app.

If the file format supports transparency (like e.g. Portable Network Graphics, in short PNG), then the background will appear transparent. This makes it easier to create image manipulations.
